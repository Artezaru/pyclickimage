pyclickimage.ClickManager#

class pyclickimage.ClickManager[source]#

Bases: object

A class to manage user clicks grouped into named categories.

This class stores 2D coordinates of points clicked on an image, grouped by string identifiers. It also handles special markers like right-clicks by storing (None, None) tuples.
